# Image or Text Recognition (OCR)

## Project Description

The Image or Text Recognition project is a Python-based application that uses Optical Character Recognition (OCR) to extract text from images. The program reads an image, converts it into grayscale, applies image preprocessing techniques, and recognizes the text using the Tesseract OCR engine. It then displays the extracted text clearly on the screen.

## Features

- Image loading
- Image preprocessing (Grayscale)
- Binary thresholding
- Optical Character Recognition (OCR)
- Display recognized text
- Display original image
- Display processed image
- Error handling for missing images

## Technologies Used

- Python
- OpenCV (cv2)
- pytesseract
- Pillow (PIL)
- Tesseract OCR Engine

## How to Run

1. Clone the repository.
2. Open the project folder in VS Code.
3. Install the required libraries:

```bash
pip install opencv-python
pip install pytesseract
pip install pillow
```

4. Install Tesseract OCR on your computer.
5. Place the image file (`sample.png`) in the project folder.
6. Open the terminal.
7. Run the program
